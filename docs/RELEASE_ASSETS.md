## Content maturity questionnaire — answer key

Dashboard → experience **Social Cafe DEV** → **Audience** (or "Content
Maturity") → Start questionnaire. Answer truthfully per our actual content:

| Question topic | Answer | Why |
| --- | --- | --- |
| Violence / weapons | **None** | No combat of any kind |
| Blood / gore | **None** | — |
| Scary / horror elements | **None** | Cosy café game |
| Romantic / sexual themes | **None** | — |
| Gambling / paid random items | **None** | No loot boxes (design rule §24) |
| Alcohol / tobacco / drugs | **None** | Coffee & pastries only |
| Strong language | **None** | All text is curated |
| Crude humor | **None** | — |
| **Free-form user content** | **No free text between players** | Compliments are a fixed whitelist; chat is Roblox's own moderated TextChatService |
| Expected rating | **All Ages / Minimal** | |

After submitting, the dashboard shows the assigned rating; screenshot it for
RELEASE_CHECKLIST. If any answer above ever changes (e.g. adding a wine item
to a menu), the questionnaire MUST be re-done before the next release.
